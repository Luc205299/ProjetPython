Bonjour, bienvenue. Ce programme à été écrit par Lucas Barthelemy et Sylvan Buhard

Python Version: 3.10

le projet contient les fichiers suivants:
pour exécuter le programme, simplement lancer main.py, et interagir avec les autres fichiers grâce à la console.
- main.py :fichier le plus important, permet d'utliliser l'interface utilisateur. Execttez seulement ce fichier pour utiliser toutes les autres fonctions
- function_books : fichier contenant toutes les fonctions relatives à la modification de tous les fichiers reliés aux données concernant les livres
- function_users.py : fichier contenant toutes les fonctions relatives à l'interaction avec les profiles d'utilisateur
- function_matrix : fichier contenant toutes les fonctions relatives à la modification de la matrice de notation utilisateurs\livres
- matrix_simi_function : fichier contenant toutes les fonctions relatives à la modification de la matrice de similarité entre utilisateurs
